# Spotify-Performance-Analysis-Dashboard- 
Spotify Performance Analytics Dashboard!!⭐️

This dashboard transforms Spotify music data into interactive insights, helping analyze song popularity, artist performance, genre distribution, and listening trends through dynamic visualizations.

🔍 Project Highlights

✅ KPI Cards for key music metrics
✅ Artist-wise song analysis
✅ Top Popular Songs identification
✅ Genre distribution insights
✅ Explicit vs Non-Explicit song analysis
✅ Monthly popularity trend tracking
✅ Interactive slicers and filters for exploration

🛠️ Tools & Technologies Used

Microsoft Excel
Power Query
Pivot Tables & Pivot Charts
Power BI
DAX Measures
Data Modeling
Data Visualization
Business Intelligence Reporting
Interactive Dashboard Design

📚 Key Learnings

Creating KPI-driven dashboards
Building efficient data models and relationships
Writing DAX measures for business metrics
Designing interactive reports for better decision-making
Transforming raw data into actionable insights
Developing storytelling skills through data visualization

📊 Insights Generated

Most popular songs and artists
